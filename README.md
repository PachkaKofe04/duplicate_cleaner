Duplicate Cleaner

Консольный инструмент для удаления дублирующихся строк из текстового файла.

Скрипт:
- читает входной файл построчно;
- обрезает лишние пробелы и убирает пустые строки;
- удаляет дубликаты, сохраняя порядок строк;
- формирует отчёт с количеством строк и дублей;
- сохраняет отчёт и список уникальных строк в выходной файл.

Формат входных данных

Обычный текстовый файл в кодировке UTF-8, по одной записи на строку, например:

text
яблоко
груша
яблоко
слива
слива
арбуз


Запуск

1) Клонировать репозиторий:

git clone <URL_РЕПОЗИТОРИЯ>
cd dupli_cleaner

2) Создаем виртуальное окружение
Через терминал: python -m venv .venv
либо в VS Code нажмите Ctrl+Shift+P, далее в строке введите: Python: Create Environment

3) Активируем окружение и устанавливаем зависимости: 
.\.venv\Scripts\Activate.ps1 (если сразу не активировалось)

4) Запускаем main.py (в терминале python main.py)/ Допустимо с указанием своих файлов: python main.py --input my_data.txt --output cleaned.txt (Где файл my_data.txt создается в корневой папке, cleaned.txt название конечного файла, который создастся самостоятельно с ответом.)
P.S. либо коротко: python main.py -i my_data.txt -o cleaned.txt

Формат ответа:
=== Отчёт ===
Всего строк:       6
Уникальных строк:  4
Удалено дублей:    2

--- Уникальные строки ---
яблоко
груша
слива
арбуз
